Desarrollo de pagina web a la asociación Afal Bolaños

Octubre 2023 - Enero 2024

Creación y mantenimiento de una pagina web a una asociación, en este cado, Afal Bolaños, la cual se dedica al cuidado de personas con Alzheimer. Este proyecto fue desarrollado en la asignatura de Comercio Electrónico, perteneciente al 4 curso del grado de Ingenieria Informática. La pagina web fue desarrollada con el CMS Prestashop, el cual es un sistema de gestión de contenidos (CMS) de código abierto para la creación de tiendas online. PrestaShop está escrito en PHP y utiliza como gestor de bases de datos MySQL. Fue una experiencia muy enriquecedora, ya que tuvimos que aprender a utilizar este CMS, además de aprender a trabajar en equipo y a gestionar el tiempo para cumplir con los plazos que nos marcaba el profesor para evaluar la asignatura y la asociación para tener la web en funcionamiento.
Coordinador del proyecto: José Jesús Castro Sánchez
- Número de integrantes del equipo: 4.





Proyecto PNOA-LiDAR

Octubre 2023 - Enero 2023

El Instituto Geográfico Nacional (IGN) participa en el Plan Nacional de Ortofotografía Aérea (PNOA), que es un proyecto conjunto entre el Gobierno y las Comunidades Autónomas, que busca obtener ortofotografías aéreas digitales de toda España, actualizadas cada 3 años, e incorpora tecnología LIDAR desde 2009. En 2009, tras una serie de pruebas piloto y la necesidad de mayor precisión en los modelos digitales del terreno, surge el proyecto PNOA-LiDAR que busca mapear España en un ciclo de 6 años con datos LiDAR. Se realizan tres coberturas: 2009-2015, 2015-2021 y 2023 en adelante (2022 en Cataluña). La zona geográfica de estudio para esta práctica de laboratorio fue la isla de La Gomera.
- Número de integrantes del equipo: 3.

Página web Footdle

Octubre 2023 - Enero 2023

Desarrollo de página web y aplicación llamada Footdle, la cual permite al usuario jugar a una variante del famoso juego Wordle, sin embargo, en este caso, relacionado con el mundo del fútbol, es decir, el usuario tratará de adivinar el nombre de un jugador de fútbol, obteniendo como pantalla inicial una serie de huecos para rellenar con letras. En caso de que el nombre introducido por el usuario sea incorrecto, se le mostrará un mensaje y no le contará como oportunidad de adivinarlo. El jugador dispone de seis oportunidades para encontrar al jugador. En caso de que una letra pertenezca al nombre y se encuentre en la posición correcta, se mostrará en verde. En caso de que la letra pertenezca al nombre pero no se encuentre en la posición correcta, se mostrará en amarillo. En caso de que la letra no pertenezca al nombre, se mostrará en rojo. En caso de que el jugador adivine el nombre, se le mostrará un mensaje de enhorabuena y se le mostrará el nombre y una imagen del jugador, junto con el club al que pertenece y la bandera de su pais.
- Número de integrantes del equipo: 2.
